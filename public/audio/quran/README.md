# Quran recitation audio — Topic 1.4 (Tilawah & Hafazan)

Drop per-ayah recitation mp3 files here. The Hafazan game plays them from
`/audio/quran/{filename}.mp3`. Until a file exists, its play button shows a
brief "audio coming soon" hint — nothing crashes.

Recommended: a clear teaching recitation (e.g. Al-Husary *Muallim*), trimmed
per ayah, normalized volume, mono 64–128 kbps mp3.

## Required files (26 total)

| File | Surah | Ayah |
|------|-------|------|
| `basmalah.mp3` | Basmalah header (shared by surahs 112/113/114/103) | — |
| `1-1.mp3` … `1-7.mp3` | Al-Fatihah | 1–7 |
| `112-1.mp3` … `112-4.mp3` | Al-Ikhlas | 1–4 |
| `113-1.mp3` … `113-5.mp3` | Al-Falaq | 1–5 |
| `114-1.mp3` … `114-6.mp3` | An-Nas | 1–6 |
| `103-1.mp3` … `103-3.mp3` | Al-'Asr | 1–3 |

Notes:
- Al-Fatihah's Basmalah **is** ayah 1, so it uses `1-1.mp3` (no separate header).
- For surahs 112/113/114/103 the Basmalah is shown as a header and uses the
  shared `basmalah.mp3`; ayah 1 of each of those is the first verse *after* the
  Basmalah (e.g. `112-1.mp3` = "Qul huwallaahu ahad").
